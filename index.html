jandroid2cloud by flosell

Browser independent client for Android2Cloud

JAndroid2Cloud is a client for Android2Cloud that opens links received from Android2Cloud in a browser of your choice. Written in Java, it runs on Windows, Mac and Linux

Dependencies

Android2Cloud, SWT and a few more

Install

1 .Download the binary package 2. extract it 3. run jandroid2cloud*.jar
